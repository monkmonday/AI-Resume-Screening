# 🤖 AI Resume Screening Tool

## 🚀 Overview
An AI-powered web application that analyzes resumes and matches them with job descriptions using Natural Language Processing (NLP).

## 🔥 Features
- Upload resume in PDF format
- Match score using TF-IDF & cosine similarity
- Skill extraction and gap analysis
- Recommendations for missing skills
- Interactive UI built with Streamlit

## 🛠 Tech Stack
- Python
- Scikit-learn
- NLTK
- Streamlit
- PyMuPDF

## 📊 Output
- Match Score (%)
- Skills Found
- Missing Skills
- Recommendations
